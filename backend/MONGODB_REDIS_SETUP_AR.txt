════════════════════════════════════════════════════════════════
📦 الخيار 1: MongoDB Atlas (موصى به - مجاني)
════════════════════════════════════════════════════════════════

1️⃣ اذهب إلى: https://www.mongodb.com/cloud/atlas/register

2️⃣ أنشئ حساب جديد (مجاني)

3️⃣ اختر:
   - FREE Tier (M0)
   - Provider: Google Cloud
   - Region: us-central1 (أو أقرب منطقة)

4️⃣ اضغط "Create Cluster" وانتظر 5 دقائق

5️⃣ إعداد الوصول:
   - اضغط "Database Access" → "Add New Database User"
   - Username: manus_user
   - Password: احفظ كلمة المرور!
   - اضغط "Add User"

6️⃣ السماح بالاتصال:
   - اضغط "Network Access" → "Add IP Address"
   - اختر "Allow Access from Anywhere" (0.0.0.0/0)
   - اضغط "Confirm"

7️⃣ احصل على Connection String:
   - اضغط "Clusters" → "Connect"
   - اختر "Connect your application"
   - انسخ الـ Connection String:
   
   mongodb+srv://manus_user:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority

   ⚠️ استبدل <password> بكلمة المرور الفعلية

8️⃣ احفظ هذا String - ستحتاجه لاحقاً!

════════════════════════════════════════════════════════════════
🔴 الخيار 2: Redis Cloud (موصى به - مجاني)
════════════════════════════════════════════════════════════════

1️⃣ اذهب إلى: https://redis.com/try-free/

2️⃣ أنشئ حساب جديد

3️⃣ اختر:
   - Fixed Plan (مجاني)
   - Provider: Google Cloud Platform
   - Region: us-central1

4️⃣ اضغط "Create subscription"

5️⃣ احصل على معلومات الاتصال:
   - اضغط على database الذي تم إنشاؤه
   - احفظ:
     * Public endpoint: redis-xxxxx.redis.cloud:xxxxx
     * Username: default
     * Password: xxxxxxxxxx

6️⃣ يمكنك أيضاً الحصول على Redis URL الكامل:
   redis://default:YOUR_PASSWORD@redis-xxxxx.redis.cloud:xxxxx

7️⃣ احفظ هذه المعلومات - ستحتاجها لاحقاً!

════════════════════════════════════════════════════════════════
📝 المعلومات التي ستحتاجها بعد الانتهاء
════════════════════════════════════════════════════════════════

احفظ هذه المعلومات وأعطيني إياها:

1️⃣ MongoDB Connection String:
   mongodb+srv://username:password@cluster.mongodb.net/manus

2️⃣ Redis Connection Info (اختر واحد):
   
   الطريقة 1 - معلومات منفصلة:
   - REDIS_HOST: redis-xxxxx.redis.cloud
   - REDIS_PORT: xxxxx
   - REDIS_PASSWORD: xxxxxxxxxx
   
   الطريقة 2 - URL كامل:
   - redis://default:password@redis-xxxxx.redis.cloud:xxxxx
